What was the first web design?

The first web design was created by Tim Berners-Lee in 1991 for the World Wide Web project. It featured a simple, text-based interface with hyperlinked documents that allowed users to navigate between pages. The website, hosted at info.cern.ch, served as an introduction to the World Wide Web, explaining how to use browsers, create web pages, and understand the HTTP and HTML protocols. This minimalist design laid the foundation for the modern web, emphasizing accessibility and functionality.
