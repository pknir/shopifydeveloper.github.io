What is the US Web Design System?

The U.S. Web Design System (USWDS) is a set of standards, guidelines, and reusable components designed to help federal agencies create accessible, user-friendly, and consistent websites. It includes design principles, templates, and tools that prioritize usability, accessibility, and performance. The system ensures government websites meet modern web standards, are mobile-responsive, and provide a seamless user experience across various devices. Created by the General Services Administration (GSA), USWDS is open-source, allowing widespread adoption and customization.

What was the first web design?
